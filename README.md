# QuantumCrypt 🔐

> A hybrid cryptographic suite developed for advanced data security.

QuantumCrypt is a comprehensive encryption and decryption tool built as my major final-year Computer Science Engineering project. It is designed to ensure data confidentiality and integrity by securing multiple formats, including plain text, images, and standard files.

## ✨ Features

* **Multi-Format Support:** Securely encrypts and decrypts text, images, and various file types.
* **Core Cryptography:** Implements robust cryptographic algorithms from scratch to manage the encryption lifecycle without relying heavily on black-box external libraries.
* **Data Integrity:** Ensures that the original file structure and content remain intact and uncorrupted post-decryption.

## 🛠️ Tech Stack

* **Languages:** Python, Flask
* **Core Concepts:** Hybrid Cryptography, Symmetric/Asymmetric Encryption, File Handling
